CSS

CSS (Cascading Style Sheets) the artist that brings web page to life with colors, fonts, layouts, and all sorts of visual flair. It's what makes websites look appealing and user-friendly

  • font-size The font-size property controls the size of the text within an element.
  • background-color The background-color property sets the background color of an element. This color fills the element's content, padding, and border areas.
  • color The color property sets the foreground color of an element's text

SEO

SEO stands for Search Engine Optimization. In simple terms, it's the practice of making your website more visible and attractive to search engines like Google, Bing, and others. The goal is to rank higher in search results for relevant keywords, which can lead to more organic (non-paid) traffic to your site.

  • < title > tag defines the title of your webpage. This title is displayed on the browser's title bar or tab and is also used as the headline in search engine results pages (SERPs).
  • <meta name="description" content="..." > The meta description provides a brief summary of your webpage's content. It's often displayed as the snippet of text below the title link in search engine results.
  • < meta name="robots" content="..." > The robots meta tag provides instructions to search engine crawlers (robots) about how to handle the page.

Git Basics

Git is a version control system that allows developers to track changes in their code, collaborate with others, and manage their project history efficiently. It’s especially useful in team environments and helps avoid issues like losing progress or overwriting each other’s work.
Benefits:

  • Keeps a history of every change
  • Makes collaboration smoother
  • Helps you revert to previous versions easily
